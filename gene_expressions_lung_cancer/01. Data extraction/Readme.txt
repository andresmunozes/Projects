## Data Source

The raw gene expression data for this project was obtained from the NCBI Gene Expression Omnibus (GEO) database under the accession number **GSE81089**.

The following files are required for the analysis and can be downloaded from the source:
* `GSE81089_FPKM_cufflinks.tsv.gz`
* `GSE81089_readcounts_featurecounts.tsv.gz`

**Source Link:** [https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE81089](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE81089)
